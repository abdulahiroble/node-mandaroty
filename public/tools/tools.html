Værktøjer som der er blevet gjort brug af
Github
Github er blevet brugt til at hoste vores projekter online så de er let tilgængelig for andre udviklere som kan klone projektet og prøve det af i deres egen computer.
Vscode
Vscode er den primære editor jeg bruger til at bygge applikationer. Den har en masse gode features såsom kode formattering, plugins, en integreret terminal etc. Kan ikke se mig selv bruge andre editors lige foreløbig.
Postman
Postman bruger jeg til at teste API funktionaliter. Det gør nemt at teste forskellige request såsom GET eller POST. Dertil har den darkmode option som for mig er vigtigt at have.
iTerm2
Dette værktøj er kun tilgængelig når jeg bruger min macbook fremfor min pc. Har aldrig rigtig været fan af windows terminal men min oplevelse har ændret sig markant siden jeg begyndte at bruge iTerm2. Den har masse fede features såsom code completion, search og så kan man customize dens udsende.
