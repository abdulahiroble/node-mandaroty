Terminal kommandoer
1. Kommandoen for at lave initialisere et nyt projekt
npm init
2. Kommandoen for at installere nodemon på din maskine
npm i nodemon
3. Kommandoen for at installere node modules
npm i
4. Kommandoen for at starte development
npm run dev
5. Kommandoen for at installere en hvilken som helst npm pakke
npm i package-name
6. Kommandoen for at fjerne node_modules
rm -rf node_modules/
7. Kommandoen for at installere express på din maskine
npm i express
