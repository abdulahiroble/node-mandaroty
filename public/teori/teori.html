Forskellige former for request
HTTP Method Endpoint What it does
GET localhost:8080/ Retrieves anime website
GET crunchyrool/anime/1 Retrieves particular anime by id
POST crunchyrool/anime/add Adds a new anime
PUT crunchyrool/anime/1 Update an existing anime
GET
GET request bruges til at hente data fra en specifik rource gennem HTTP protokollen.
POST
POST request bruges til at sende data til en server for enten at oprette eller opdatere en rource.
PUT
PUT request bruges til at sende data til en server for at oprette eller opdatere en rource.
Forskellen mellem POST og PUT er at PUT altid vil producere det samme resultat hvorimod POST vil oprette den samme resource hvis du kalder POST flere gange.
PATCH
PATCH request bruges til at modificere dele af en resource og bør ikke sammenlignes med PUT som er en komplet repræsentation af en resource.
DELETE
DELETE request sletter en given resource, eksempelvis ud fra dens id.
